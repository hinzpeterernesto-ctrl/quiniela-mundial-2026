# ⚽ Quiniela Mundial 2026

Quiniela para jugar con amigos durante el Mundial 2026. Se instala como app en tu celular.

## 🚀 Cómo publicar en Vercel (GRATIS)

### Paso 1: Sube el proyecto a GitHub
1. Ve a [github.com](https://github.com) y crea una cuenta si no tienes
2. Crea un nuevo repositorio (botón verde "New")
3. Ponle nombre: `quiniela-mundial-2026`
4. Sube todos los archivos de esta carpeta al repositorio

> **Tip rápido**: Puedes arrastrar los archivos directamente en la página de GitHub

### Paso 2: Publica en Vercel
1. Ve a [vercel.com](https://vercel.com) y regístrate con tu cuenta de GitHub
2. Haz clic en **"Add New → Project"**
3. Selecciona tu repositorio `quiniela-mundial-2026`
4. **Framework Preset**: selecciona **Vite**
5. Haz clic en **Deploy**
6. ¡Listo! En ~1 minuto tendrás tu URL (ej: `quiniela-mundial-2026.vercel.app`)

### Paso 3: Instalar como app en el celular
1. Abre el link en el navegador de tu celular
2. **iPhone**: toca el ícono de compartir (cuadrito con flecha) → "Agregar a pantalla de inicio"
3. **Android**: toca los 3 puntitos → "Agregar a pantalla de inicio" o te saldrá automático

### Paso 4: Comparte con tus amigos
- Manda el link por WhatsApp
- Cada quien lo abre y lo instala en su celular
- ¡Listo para jugar!

## 📋 Sistema de puntaje
| Acierto | Puntos |
|---------|--------|
| Marcador exacto | +5 |
| Resultado correcto (quién gana/empate) | +3 |
| Goles de un equipo | +1 |
| Sin acierto | 0 |

## ⚠️ Nota importante
Los datos se guardan en el **localStorage del navegador** de cada dispositivo. Esto significa que cada celular/computadora tiene sus propios datos. Si todos juegan desde el mismo dispositivo, perfecto. Si cada quien juega desde su celular, necesitarán que UNA persona sea el "admin" que mete los resultados reales y que todos vean la tabla de posiciones en un mismo dispositivo.
